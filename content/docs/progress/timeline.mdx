## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm install framer-motion lucide-react
```

<Step>Update `tailwind.config.js`</Step>

Add the following to your tailwind.config.js file.

```json
module.exports = {
  theme: {
    extend: {
    }
  }
}
```

<Step>Run the following command</Step>

It will create a new file `timeline.tsx` inside the `components/mage-ui/progress` directory.

```bash 
mkdir -p components/mage-ui/progress && touch components/mage-ui/progress/timeline.tsx
```

<Step>Paste the code</Step> 

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/progress/timeline.tsx

```
