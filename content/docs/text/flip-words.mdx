## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm i motion clsx tailwind-merge
```

<Step>Add util file</Step>

lib/utils.ts

```json
import { ClassValue, clsx } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}

```

<Step>Run the following command</Step>

It will create a new file `flip-words.tsx` inside the `components/mage-ui/text` directory.

```bash 
mkdir -p components/mage-ui/text && touch components/mage-ui/text/flip-words.tsx
```

<Step>Paste the code</Step> 

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/text/flip-words.tsx

```
