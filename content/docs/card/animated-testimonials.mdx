## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm i motion clsx tailwind-merge
```

<Step>Add util file</Step>

lib/utils.ts

```json
import { ClassValue, clsx } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}

```

<Step>Run the following command</Step>

It will create a new file `animated-testimonials.tsx` inside the `components/mage-ui/card` directory.

```bash
mkdir -p components/mage-ui/card && touch components/mage-ui/card/animated-testimonials.tsx
```
