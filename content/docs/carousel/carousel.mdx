## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm install framer-motion lucide-react
```

<Step>Update `tailwind.config.js`</Step>

Add the following to your tailwind.config.js file.

```json
module.exports = {
  theme: {
    extend: {
    }
  }
}
```

<Step>Run the following command</Step>

It will create a new file `carousel.tsx` inside the `components/mage-ui/carousel` directory.

```bash 
mkdir -p components/mage-ui/carousel && touch components/mage-ui/carousel/carousel.tsx
```

<Step>Paste the code</Step> 

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/carousel/carousel.tsx

```
</Steps>
