## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm install framer-motion lucide-react
```

<Step>Run the following command</Step>

It will create a new file `globe.tsx` inside the `components/mage-ui/icon` directory.

```bash 
mkdir -p components/mage-ui/icon && touch components/mage-ui/icon/globe.tsx
```

<Step>Paste the code</Step> 

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/icon/globe.tsx

```
</Steps>
