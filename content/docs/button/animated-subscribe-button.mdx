## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm install framer-motion lucide-react
```

<Step>Update `tailwind.config.js`</Step>

Add the following to your tailwind.config.js file.

```json
module.exports = {
  theme: {
    extend: {
    }
  }
}
```

<Step>Run the following command</Step>

It will create a new file `animated-subscribe-button.tsx` inside the `components/mage-ui/button` directory.

```bash 
mkdir -p components/mage-ui/button && touch components/mage-ui/button/animated-subscribe-button.tsx
```

<Step>Paste the code</Step> 

Open the newly created file and paste the following code:

```jsx file=<rootDir>/mage-ui/button/animated-subscribe-button.tsx

```
</Steps>
